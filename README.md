# Airline-management-system-
Python-based Airline Management System for managing clients, airlines, and flights with GUI, storage, and testing modules.
## Team Roles

- GUI / UX Designer – Build the user interface - Rieme
- CRUD Programmer – Implement create, read, update, and delete functions - Abdel
- Storage Programmer – Handle JSON data loading and saving
- Flight Logic Programmer – Manage flight validation and logic
- Tester – Write unit tests and identify bugs - Mohamed
- Project Manager / Documentation – Manage documentation and project organisation - Amber

## Development Workflow
Each team member works on a separate Git branch based on their role. Changes are later merged into the main branch using pull requests.

Branches include:
- gui
- crud
- storage
- flight_logic
- testing
- documentation

## How to Run the Project
1. Clone the repository
2. Install required dependencies
3. Run the main program
